Usage
-----

```
cmf new data [options] [<workingDir>]
```

### Arguments

Name | Description
---- | -----------

### Options

Name | Description
---- | -----------
`--version <version>` | Package Version [default: 1.1.0]
`--businessPackage <businessPackage>` | Business package where the Process Rules project should be built [default: ]
