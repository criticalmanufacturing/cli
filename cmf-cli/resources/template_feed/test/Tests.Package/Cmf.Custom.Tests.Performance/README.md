# CM MES Performance Tests

CM MES Performance Tests implemented using [Grafana k6](https://github.com/grafana/k6/) and leveraging the TypeScript Light Business Objects (LBOs).

For detailed guidance on how to set up, create, and run performance tests, refer to our [**Developer Portal**](https://developer.criticalmanufacturing.com/explore/guides/customizations/test/performance/).

## Quick Start

If you just want to run the tests:

```bash
npm start Tests/get_site.ts
```
