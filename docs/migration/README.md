# Migration Guides

This directory contains comprehensive migration guides for the Critical Manufacturing CLI.

## Available Migration Plans

### [.NET 8 to .NET 10 Migration Plan](./dotnet10-migration-plan.md)

A complete, production-ready migration plan for upgrading from .NET 8 to .NET 10.

**Includes:**
- ✅ Breaking changes analysis with risk assessment
- ✅ Step-by-step migration strategy with checkpoints
- ✅ Code modernization opportunities (C# 13 features)
- ✅ Tooling and ecosystem readiness (SDK, CI/CD, Docker)
- ✅ Comprehensive risk assessment
- ✅ Detailed validation checklists (pre, during, post-migration)
- ✅ Rollback strategies
- ✅ Performance monitoring guidelines
- ✅ 3-4 week timeline with resource requirements

**Key Sections:**
1. Breaking Changes Analysis - Identify likely issues
2. Step-by-Step Migration Strategy - Safe upgrade sequence
3. Code Modernization Opportunities - New .NET 10 features
4. Tooling and Ecosystem Readiness - SDK, CI/CD, Docker updates
5. Risk Assessment - High-risk areas and mitigation
6. Validation Checklist - Complete verification process
7. Timeline and Resources - Estimated effort
8. Success Criteria - Clear metrics for success/failure

**Status:** Ready for use  
**Last Updated:** 2026-01-30

## How to Use These Guides

1. **Read the entire plan** before starting any migration
2. **Customize the plan** to your specific needs and timeline
3. **Use the checklists** to track your progress
4. **Follow the rollback strategy** if issues are encountered
5. **Document your results** for future reference
