# ls

<!-- BEGIN USAGE -->

Usage
-----

```
cmf ls [options] [<workingDir>]
```

### Arguments

Name | Description
---- | -----------
`<workingDir>` | Working Directory [default: .]

### Options

Name | Description
---- | -----------
`-r, --repo, --repos <repos>` | Repositories where dependencies are located (folder)
`-?, -h, --help` | Show help and usage information
